# Currency Converter Android App

A modern Android Currency Converter built using Java and XML in Android Studio.

## Features
- Convert currencies instantly
- User-friendly interface
- Real-time exchange rates
- Supports multiple currencies
- Lightweight and fast

## Supported Currencies
- USD (United States Dollar)
- INR (Indian Rupee)
- EUR (Euro)
- KRW (South Korean Won)
- JPY (Japanese Yen)

## Technologies Used
- Java
- XML
- Android Studio
- Spinner
- CardView

## Installation
1. Clone repository
   
    git clone:- https://github.com/yourusername/Currency-Converter-Android-App.git

2. Open Android Studio
  
3. Sync Gradle
   
4. Run on Emulator or Android Device
